About This Website

This website is build as a project for GCD - Internet Authoring Assignment. Thinking about the main protagonist for my website I needed to build for Internet Authoring module I decided to make a website about a person that made my childhood filled with laughter. I was in love with Charlie Chaplin's movies since I was a kid and not only that this movies would made me laugh, I was also trying do dig for a message that Charlie Chaplin was trying to point out in every movie that he created.

This website is dedicated to Charlie Chaplin.
